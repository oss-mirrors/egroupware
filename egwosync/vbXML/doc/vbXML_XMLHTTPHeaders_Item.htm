Item Method

See Also   Example   Applies To

Returns a string which is the HTTP header in the collection by position.

Syntax

object.Item (index)

Part Description
object An object expression that evaluates to an XMLHTTPHeaders object.
index Required. An expression that specifies the position of a string which is a HTTP header in the collection. A Long, index must be a number from 1 to the value returned by the Count property.

Remarks

If the value provided as index doesn’t match any existing item in the collection, an error occurs.

The Item method is the default method for the object. Therefore, the following lines of code are equivalent:

Print MyObject(1)
Print MyObject.Item(1)
