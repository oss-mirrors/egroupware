ParseString Method

See Also   Example   Applies To

The passed string with the contents of the XML document is parsed and may check the document is well-formed.

Syntax

object.ParseString (Data, CheckWellFormed)

Part Description
object An object expression that evaluates to an XMLSAXParser object.
Data Required. A string with the contents of the XML document.
CheckWellFormed Optional. A boolean that defines whether the parser should check the XML is "well formed". Defaults to True, i.e. will check the document is well-formed.

Remarks

The default behaviour of this method is to check the document is well formed. No validation against a DTD can be carried out using this object.
