XML_Errors Enumerated List

See Also   Example   Applies To

This enumerated list contains the additional errors that can be thrown by vbXML.dll.

vbXML_ERROR_WININETERROR

Error Number -2146875826 : WININET.dll error.

vbXML_ERROR_HTTPERROR

Error Number -2146875825 : HTTP Error.

vbXML_ERROR_MALFORMEDXML

Error Number -2146875824 : Malformed XML.

vbXML_ERROR_UNSUPPORTED

Error Number -2146875823 : Unsupported Feature of XML Standard.

vbXML_ERROR_MALFORMEDDTD

Error Number -2146875822 : Malformed DTD.

vbXML_ERROR_NODTDTOVALIDATEWITH

Error Number -2146875821 : No DTD to Validate File With.

vbXML_ERROR_INVALIDXML

Error Number -2146875820 : Invalid XML.

vbXML_ERROR_UNKNOWNREQUESTTYPE

Error Number -2146875819 : Unknown Request Type.

vbXML_ERROR_UNKNOWNPROTOCOLTYPE

Error Number -2146875818 : Unknown Protocol Type.

vbXML_ERROR_ATTRIBUTENOTFOUND

Error Number -2146875817 : Thrown by the XMLAttributes.GetAttributeByName method when an attribute of the passed name is not found.

vbXML_ERROR_TAGNOTFOUND

Error Number -2146875816 : Thrown by the XMLDOMNode.GetTagByName method when a tag of the passed name is not found.

vbXML_ERROR_DTDELEMENTNOTFOUND

Error Number -2146875815 : Thrown by the XMLDTDElements.GetElementByName method when a DTD element of the passed name is not found.

vbXML_ERROR_UNDEFINEDNAMESPACE

Error Number -2146875814 : Namespace is not defined in this document at this level.

vbXML_ERROR_PORTOUTOFRANGE

Error Number -2146875813 : Thrown by the XMLHTTPCall.HostPort property when the port is out of range; must be in the range 1 to 65535.
