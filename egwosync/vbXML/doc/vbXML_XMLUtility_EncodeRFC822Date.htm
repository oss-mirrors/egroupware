EncodeRFC822Date Method

See Also   Example   Applies To

Returns a string representing an RFC822 date.

Syntax

object.EncodeRFC822Date (DateToEncode,TimeZone)

Part Description
object An object expression that evaluates to an XMLRPCUtility object.
DateToEncode Required. A date representing the date to be converted.
TimeZone Required. A string representing the three letter timezone code e.g. GMT.

Remarks

RFC822 dates are in the form "Tues, 12 Mar 1945 12:34:56 GMT".
