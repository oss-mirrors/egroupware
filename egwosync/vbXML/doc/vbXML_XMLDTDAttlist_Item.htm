Item Method

See Also   Example   Applies To

Returns an XMLDTDAttribute object from the collection by position.

Syntax

object.Item (index)

Part Description
object An object expression that evaluates to an XMLDTDAttlist object.
index Required. An expression that specifies the position of an XMLDTDAttribute object of the object. A Long, index must be a number from 1 to the value returned by the Count property.

Remarks

If the value provided as index doesn’t match any existing XMLDTDAttribute object of the object, an error occurs.

The Item method is the default method for the object. Therefore, the following lines of code are equivalent:

Print MyObject(1).Name
Print MyObject.Item(1).Name
