HTTP_Request_Types Enumerated List

See Also   Example   Applies To

This enumerated list contains the two types of request supported, the "GET" request and the "POST" request.

HTTP_GET

Value 1 - An HTTP "Get" request that just fetches the required resource.

HTTP_POST

Value 3 - An HTTP "Post" request that sends data to the remote server and fetches the response. Used in XML-RPC, XML-SOAP etc.
