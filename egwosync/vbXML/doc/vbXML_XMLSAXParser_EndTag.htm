EndTag Event

See Also   Example   Applies To

Fired when an end tag is found in the XML document.

Syntax

Private Sub object_EndTag (Name , StartPostion)

Part Description
object An object expression that evaluates to an XMLSAXParser object.
Name Required. A string with the name of the end tag.
StartPosition Required. A long integer with the start position in bytes of the tag in the XML document.
