Value Property

See Also   Example   Applies To

Read-Only. A string containing the value of the base node - whether it be the name or some data.

Syntax

object.Value

The object placeholder represents an XMLDOMParser object.

Remarks

  • If the Node is of type XML_SPECIAL the value is the whole tag.
  • If the Node is of type XML_TAG the value is the name of the tag, in the following example the value will be 'ExampleTag':
    <ExampleTag ExampleName1="Example Value" ExampleName2="Another Example Value" />
  • If the Node is of type XML_VALUE the value is the whole string.
