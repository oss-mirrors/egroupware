XML_DTDAttributeModifiers Enumerated List

See Also   Example   Applies To

This enumerated list contains rules defining how the attributes in a DTD <ATTLIST ...> declaration can be modified.

XMLDTD_NODTDATTRIBUTEMODIFIER

Value 0 - No modifier.

XMLDTD_REQUIRED

Value 1 - #REQUIRED - The attribute value must be specified with the element.

XMLDTD_IMPLIED

Value 2 - #IMPLIED - The attribute value is unspecified, to be determined by the application.

XMLDTD_FIXED

Value 3 - #FIXED - The attribute value is fixed and cannot be changed by the user.
