GetErrorCode Method

See Also   Example   Applies To

Returns a string with a text description of the error code.

Syntax

object.GetErrorCode (ErrorNumber)

Part Description
object An object expression that evaluates to an XMLUtility object.
ErrorNumber Required. A long integer (type Long) which is the error number.

Remarks

When a special vbXML error is thrown the following information is passed to the calling application:

  • The error code e.g. vbXML_ERROR_MALFORMEDXML
  • The method name e.g. "vbXML.dll#HTTPCall.cls#SubmitRequest"
  • A comment about the error e.g. "WinInet.dll Error Number 12345 : Server Sleeping off Hangover"

An example of the code fragment needed is below:

Dim X as New XMLUtility ' The Utility object

Print X.GetErrorCode(vbXML_ERROR_MALFORMEDXML)
