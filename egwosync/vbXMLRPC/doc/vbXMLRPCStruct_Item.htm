Item Method

See Also   Example   Applies To

Returns a specific XMLRPCMember of the object by position.

Syntax

object.Item (index)

Part Description
object An object expression that evaluates to an XMLRPCStruct object.
index Required. An expression that specifies the position of an XMLRPCMember member of the object. A Long, index must be a number from 1 to the value returned by the Count method.

Remarks

If the value provided as index doesn’t match any existing XMLRPCMember of the object, an error occurs.

The Item method is the default method for the object. Therefore, the following lines of code are equivalent:

Print MyObject(1)
Print MyObject.Item(1)
