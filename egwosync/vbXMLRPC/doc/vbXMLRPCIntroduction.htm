vbXMLRPC.dll Introduction

Welcome to the vbXMLRPC.dll documentation, this is a work in progress and will evolve in response to the needs of users. If you have any comments or suggestions please use the projects forums to comment.

vbXMLRPC.dll provides a COM implementation of a client interface to web services implementing the XML-RPC standard. It is released under the GNU Library or Lesser General Public License (LGPL).

This page contains install notes, release notes, a description of the XML-RPC standard, a getting started guide to using vbXMLRPC.dll, a concise reference, a collection of examples and finally, comments from the developer on licensing and other stuff.
