AddDouble Method

See Also   Example   Applies To

Adds an XMLRPCValue object containing a double to the object.

Syntax

object.AddDouble (value)

Part Description
object An object expression that evaluates to an XMLRPCParams object.
value Required. Double to be appended to the XMLRPCParams object.

Remarks

An example of the code fragment needed is below:

Dim X as New XMLRPCRequest ' The request to be have params added to it

X.Params.AddDouble 1.2345
