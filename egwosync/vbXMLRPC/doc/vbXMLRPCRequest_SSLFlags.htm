SSLFlags Property

See Also   Example   Applies To

A value from the XMLRPC_SSLTypes enumerated list specifying whether the call is to be made by SSL, and if it is to be made by SSL, what rules are applied.

Syntax

object.SSLFlags

The object placeholder represents an XMLRPCRequest object.

Remarks

See the topic on the XMLRPC_SSLTypes enumerated list for more details. The default value is XMLRPC_SSL_NONE, i.e a standard HTTP call.
