XMLRPCUtility Object

See Also   Example   Properties   Methods   Events

The XMLRPCUtility object contains utility functions for getting meaningfull strings from passed numbers.

Class Diagram

Remarks

Use this object to get meaningful strings explaining the following: vbXMLRPC.dll errors, HTTP Status Codes, Variant Types and XML-RPC value types.

Dim X As New XMLRPCUtility
Dim Y As String

Y = X.GetXMLRPCType(XMLRPC_STRING)
