AddBoolean Method

See Also   Example   Applies To

Adds an XMLRPCValue object containing a boolean to the object.

Syntax

object.AddBoolean (value)

Part Description
object An object expression that evaluates to an XMLRPCArray object.
value Required. Boolean to be appended to the XMLRPCArray object.

Remarks

An example of the code fragment needed is below:

Dim X as New XMLRPCArray ' The Array to be added to

X.AddBoolean True
