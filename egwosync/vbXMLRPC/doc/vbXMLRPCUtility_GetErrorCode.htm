GetErrorCode Method

See Also   Example   Applies To

Returns a string with a text description of the error code.

Syntax

object.GetErrorCode (ErrorNumber)

Part Description
object An object expression that evaluates to an XMLRPCUtility object.
ErrorNumber Required. A long integer (type Long) which is the error number.

Settings

See the XMLRPC_Errors enumerated list.

Remarks

When a special vbXMLRPC error is thrown the following information is passed to the calling application:

  • The error code e.g. vbXMLRPC_ERROR_HOSTNAMENOTSET
  • The method name e.g. "vbXMLRPC.dll#Request.cls#Submit"
  • A comment about the error e.g. "WinInet.dll Error Number 12345 : Server Sleeping off Hangover"

An example of the code fragment needed is below:

Dim X as New XMLRPCUtility ' The Utility object

Print X.GetErrorCode(vbXMLRPC_ERROR_HOSTNAMENOTSET)
